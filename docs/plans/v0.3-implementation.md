## Step 1 設計

- `SOURCE_NAMES`: `cur` | `ce` | `logs` | `auto`（`auto` は cur→ce のみ、logs は含めない）
- `today` コマンド: `--source logs` 必須（他 source はエラー）
- `ResolvedSourceName`: `cur` | `ce` | `logs`
- config `[logs]`: `log_group`（必須）、`region`（省略時は aws.region）
- レポートヘッダ: `source: CloudWatch Logs estimate`
- 概算コスト: `~$` + `estimated cost, not billing data` 注記
- principal: `identity.arn` でフィルタ（`self`/`arn`/`role`/`all`）。`--principal-tag` は logs 非対応（明示エラー）
- 本文フィールド（`inputBodyJson`, `outputBodyJson` 等）は SELECT に含めない
